It all started when my coworkers made scripts that only worked for Mac. After that I had to switch. Windows was not an option anymore.

Looking back I say I like the hardware of the Mac way more then the Windows machines we used to get. Long battery life, good screen, good speakers, etc.
Nothing to complain about. 

But working with the software has some desires and quirks that made me itchy. So I started looking for solutions for myself.

Quickly I found that other people on the internet are exactly in the same boat.

- [Alt+tab](https://github.com/lwouis/alt-tab-macos)
  - This make the Alt Tabbing the same as on Windows, there is no option to make it flip between all open windows including the same application that is open. 
    An example being Intelij. Lets say you have 5 windows open in Mac. You need to use <kbd>\`</kbd> + <kbd>Command</kbd> in order to switch between open windows of the same kind.
    That means that if you use <kbd>Alt</kbd> + <kbd>Tab</kbd> you are never going to go to the same application but then another window. *Alt+Tab* fixes this issue.
- [Rectangle](https://github.com/rxhanson/Rectangle)
