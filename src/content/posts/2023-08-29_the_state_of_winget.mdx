## Winget.run

<Image src={WingetDotRun} alt="The website winget.run" format="avif" />

You might have already seen this page before if you are a winget user (hopefully). But if not, let me introduce you to [Winget.run](https://winget.run).
This website is basically a UI search for Winget. But that is not all there are a plethora of tools available to use for Winget.
Not everyone is a CLI only fan, including me, that's why I like Winget.run. But after learning about Chocolatey at work (that's the only tool we can use there). I started to wonder, what else is available for Windows?

__Is there not something else that can combine everything?__

## Winget UI

One of those tools is [Winget UI](https://github.com/marticliment/WingetUI). Not only does it function as a Winget package manager UI, but also as a UI for Chocolatey and Scoop. So you can search new software, update and manage all in the same tool. Yes you read it correctly, all three tools combined, and even more. They also have integrations for PIP, NPM packages, Powershell and .NET Tool.

Here is a picture of the tool in action. On the Source column you can see where the package is actually being downloaded from, together with the version and the package ID.

<Image src={WingetUI} alt="The Winget UI Sofware Search" format="avif" />

If you want to use Winget UI yourself then you're in luck!
Installing the Winget UI tool is really simple with Winget itself, by just using:

```bash
winget install -e --id SomePythonThings.WingetUIStore
```

And you can already start using it right away. And even comes with dark mode ğŸŒš
